# EzyGoa Taxi & Rental Website

Modern, SEO-optimized website for EzyGoa Taxi Services in Goa, India.

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui

## Development
```bash
npm install
npm run dev
```

## Production Build
```bash
npm run build
npm start
```

## Deployment
Deployed on Hetzner via Easypanel using Docker.

## Features
- 30+ vehicles fleet (taxi + self-drive)
- SEO optimized pages
- WhatsApp integration
- Mobile-first design
- Airport transfer pricing
- Tour packages (North & South Goa)
- Outstation routes
- Self-drive rental catalogue
